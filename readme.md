**Webpack Starter Kit**

**How to use**
- Clone this repository
- Nodejs installed (npm install webpack -g)
- Run npm install
- Run npm start
- Open localhost:3000 in your browser
- You can delete the dev folder or modify it to your own needs.
- For the production mode run npm build.
