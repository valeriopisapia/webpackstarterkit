**Webpack Starter Kit**

**How to use**
- Clone this repository
- Nodejs installed (_npm install webpack -g_)
- Run _npm install_
- Run _npm start_
- Open localhost:3000 in your browser
- You can delete the dev folder or modify it to your own needs.
- For the production mode run _npm run build_.
